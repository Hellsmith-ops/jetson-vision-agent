# Jetson Vision Agent

AI-powered computer vision and agent workflow project built on NVIDIA Jetson Orin Nano.

This project explores computer vision, machine learning, and agentic AI workflows using Jetson, Docker, and modern AI tooling. The long-term goal is to evolve from basic vision tasks into intelligent systems capable of perception, reasoning, and tool use.

## Project Goals

- Learn computer vision workflows on Nvidia Jetson
- Build AI-powered vision pipelines
- Explore LLM and agent integration
- Develop reproducible workflows with Docker
- Practice software engineering and documentation habits

## Hardware

- NVIDIA Jetson Orin Nano Super Dev Kit
- JetPack 6.2 (L4T R36.5)
- NVMe storage
- USB-C networking / SSH workflow

## Tech Stack

- Python
- OpenCV
- PyTorch
- Ultralytics (YOLO)
- Docker
- JupyterLab
- Git / GitHub
- VS Code Remote SSH
